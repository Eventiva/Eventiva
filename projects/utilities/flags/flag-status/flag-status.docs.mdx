A component for a FlagStatus entity, supports serialization and de-serialization of FlagStatus objects and can be used from
both backend and frontend.

## Get started

Install and use the entity to create a new instance:

```ts
const flagStatus = FlagStatus.from({
  name: 'hello'
});
```
