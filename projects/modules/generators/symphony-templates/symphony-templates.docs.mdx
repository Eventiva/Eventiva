Templates for building aspects for Symphony platforms:

```ts
// my-env.bit-env.ts
class MyEnv extends HarmonyEnv {
  generators() {
    return SymphonyTemplates({
      name: 'checkpoint-platform',
    })
  }
}
```

## Fork for your own platform

If you wish to customize Aspects for your own platform, fork this component and change the needed files:

```
bit fork bitdev.symphony/generators/symphony-templates
```
